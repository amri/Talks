Console

Code Description
docker info get details of the docker host
docker images list all images
docker ps list running container
docker ps -a list all container
docker ps -a list all container
docker inspect <container_id> get settings for a container
docker logs <container_id> view logs for a container

Creation (1)

Code Description
docker build -t <image_name> . build an <image_name> using a Dockerfile in the current directory
docker run -d -p <host_port>:<guest_port>
--link <container_name>:<alias>
-e <env_var>
-v <host_volume>:<guest_volume>
--name <container_name>
<image_name>: <tag> 		run an image as a container (-d: detached/-it: interactive)

Creation (2)

Code Description
docker exec -it <container_id> bash run bash on a container
docker kill <container_id> stop running container immediately
docker stop <container_id> stop running container gracefully
docker rm <container_id> remove stopped container
docker start <container_id> start a stopped/killed container
docker rmi <image_id> remove image

Change Management

Code Description
docker commit -m "comment"
-a "author_name"
<container_id>
<image_name>:<tag> 		Commit changes made to a container as an image
docker save -o <save_image_to_path> <image_name> Save an image to a file
docker load -i <path_to_image_file> Load an image from a file
docker tag <image_id>
<image_name> 		Rename an image or put a tag on it

Docker Compose

  • Previously called Fig
  • Managing multiple containers using docker-compose.yml
  • Orchestrating multiple containers
Code Description
docker-compose up -d Build and running (detached) all containers
docker-compose kill/stop Kill/Stop all containers
docker-compose logs List logs from all containers
